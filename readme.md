# 美居H5

## Build Setup

### clone repo 

```
git clone git@gitee.com:koooo/meiju-h5.git
cd meiju-h5
git checkout dev
```

### Install dependencies
```
npm install
```

### Serve with hot reload at localhost:8080
```
npm run serve
```

### Build for production with minification
```
npm run build
```

### Auto-fix esLint
```
npm run lint
```
